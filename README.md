# Physio Exam Prep

Uma plataforma web responsiva desenvolvida para auxiliar estudantes de Fisioterapia na revisão de conteúdos relacionados à Biossegurança, Gestão de Riscos e Primeiros Socorros.

O projeto reúne conteúdo teórico, flashcards interativos e um simulado acadêmico para tornar o processo de revisão mais eficiente, organizado e acessível.

---

## Sobre o Projeto

A preparação para avaliações acadêmicas exige a revisão de uma grande quantidade de conteúdo em um curto período de tempo. O Physio Exam Prep foi criado com o objetivo de transformar esse processo em uma experiência mais prática, visual e interativa.

A plataforma apresenta conceitos fundamentais da área da saúde por meio de explicações simplificadas, recursos de memorização ativa e exercícios para autoavaliação.

---

## Funcionalidades

### Conteúdo Teórico

O material aborda temas importantes para a formação acadêmica em Fisioterapia, incluindo:

* Biossegurança e Legislação
* Classificação dos Riscos Ambientais
* Higienização das Mãos
* Precauções e Isolamento
* Gestão de Resíduos em Serviços de Saúde (PGRSS)
* Ergonomia, Ética e Responsabilidade Profissional

### Primeiros Socorros e Emergências

Conteúdos voltados para o atendimento inicial em situações de urgência e emergência:

* Protocolo XABCDE
* Ressuscitação Cardiopulmonar (RCP)
* Obstrução das Vias Aéreas por Corpo Estranho (OVACE)
* Emergências Clínicas
* Emergências Traumáticas
* Atuação da Fisioterapia em Situações Críticas

### Flashcards Interativos

Ferramenta de memorização ativa que permite ao estudante testar seus conhecimentos e reforçar conceitos importantes de forma rápida e dinâmica.

### Simulado Acadêmico

Sistema de avaliação com:

* Questões de múltipla escolha
* Questões dissertativas
* Correção automática das questões objetivas
* Exibição das respostas esperadas nas questões dissertativas
* Feedback imediato de desempenho

### Design Responsivo

Interface desenvolvida com abordagem mobile-first, garantindo uma boa experiência de uso em:

* Smartphones
* Tablets
* Notebooks
* Computadores desktop

## Como Executar o Projeto

Clone o repositório:

```bash
git clone https://github.com/seu-usuario/physio-exam-prep.git
```

Acesse a pasta do projeto:

```bash
cd physio-exam-prep
```

Abra o arquivo `index.html` em seu navegador ou utilize a extensão **Live Server** no Visual Studio Code.
